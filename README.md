# 인플루언서 인게이지먼트 핏 분석 대시보드

인플루언서 컨택 전에, 브랜드와 잘 맞는 인스타그램 계정을 데이터 기반으로 골라내는 도구입니다.

## 어떻게 동작하나요

1. **로컬 PC에서 수집** — `scraper/collect.py` 가 인스타그램 **공개 프로필 정보**(팔로워 수,
   최근 게시물 좋아요/댓글 수, 소개글 등)를 로그인 없이 조회해 `data/influencers.csv` 에 저장합니다.
2. **GitHub에 푸시** — `PUSH-Update.bat` 을 실행하면 수집한 데이터가 GitHub 저장소에 올라갑니다.
3. **Streamlit Cloud에서 조회** — 클라우드 대시보드는 이 CSV를 읽어서 순위표·차트·상세 프로필을 보여줍니다.

수집은 항상 로컬에서, 조회는 클라우드에서 — 기존 META 광고 레퍼런스 대시보드와 동일한 구조입니다.

## ⚠️ 꼭 읽어주세요 — 수집 관련 주의사항

- 이 도구는 **로그인하지 않고 공개된 정보만** 조회합니다. Instagram 이용약관상 자동 수집 자체가
  회색 지대이므로, **소량(하루 수십 개 이하)** 으로만 사용하고 계정당 8~20초의 지연을 유지하세요.
- 비공개(private) 계정은 팔로워 수 외의 정보(게시물 좋아요/댓글)를 가져올 수 없습니다.
- 요청이 몰리면 일시적으로 429(요청 제한) 응답이 올 수 있습니다. 이 경우 수집이 자동으로
  중단되니, 몇 시간 후 다시 시도하세요. 짧은 시간에 반복 재시도하지 마세요.
- 절대 다량의 계정을 한 번에 수집하거나, 여러 사람이 동시에 같은 IP로 돌리지 마세요 — 계정
  일시 제한(rate limit)의 원인이 됩니다.
- 대시보드의 "계정 추가" 기능은 **로컬 실행 시에만** 동작합니다. Streamlit Cloud에서는 조회만
  가능하도록 막혀 있습니다.

## 로컬에서 실행하기

1. `START-Windows.bat` 더블클릭 — 최초 실행 시 자동으로 필요한 패키지를 설치합니다.
2. `config.json` 의 `influencers` 목록에 확인하고 싶은 인스타그램 아이디를 추가하세요.
   (또는 대시보드 사이드바에서 "계정 추가"로 하나씩 추가해도 됩니다.)
3. `COLLECT-Windows.bat` 을 실행해 수집합니다.
4. 대시보드에서 결과를 확인하세요.
5. 클라우드에도 반영하려면 `PUSH-Update.bat` 을 실행하세요.

## 핏 스코어는 어떻게 계산되나요

`dashboard/scoring.py` 에서 계산하며, 아래 네 가지를 종합합니다.

| 기준 | 비중 | 설명 |
|---|---|---|
| 인게이지먼트율 | 50% | (평균 좋아요+댓글) / 팔로워 — 최근 게시물 최대 12개 기준 |
| 브랜드 키워드 매칭 | 25% | 소개글·카테고리에 브랜드 키워드가 있으면 가산 |
| 게시 활발도 | 15% | 최근 게시물 간 평균 업로드 간격 |
| 팔로워 구간 적합 | 10% | 사이드바에서 선택한 타겟 구간(나노/마이크로/매크로/메가)과 일치하는지 |

브랜드 키워드는 대시보드 사이드바에서 바로 수정할 수 있고, `config.json` 에 저장됩니다.

## 파일 구조

```
config.json              브랜드 키워드, 수집 대상 계정 목록, 요청 지연 시간
scraper/instagram_public.py  공개 프로필 조회 함수
scraper/collect.py       수집 실행 스크립트 (CLI)
dashboard/scoring.py     핏 스코어 계산 로직
dashboard/app.py         Streamlit 대시보드 화면
data/influencers.csv     수집 결과 (자동 생성)
```
